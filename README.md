# Steel Monsters

It's a 2D battle game between tanks

# Building

- Generate configuration
    ```sh
    autoreconf -i
    ```

- Generate makefiles
    ```sh
    ./configure
    ```
- Build program
    ```sh
    make
    ```

# Installing
```sh
make install
```
